# 🎯 Movie Recommendation System (Java)

This is a simple Java project that recommends movies based on similarity between their genres. It is a basic content-based recommendation system made using core Java.


## 💡 How It Works

- Each movie has a list of genres (like sci-fi, action, drama, etc.)
- The program compares the selected movie with all other movies
- It checks how many genres are common between them
- Based on similarity score, it recommends top 3 movies
